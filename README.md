## 🚀 Phase Movement Tracker

**Description:**
Phase Movement Tracker is a web application that helps track students' progress across different learning phases. It allows mentors to monitor performance and manage phase transitions in a structured and transparent way.

---

## 🔑 Features

* Students can view their current phase and progress
* Mentors can assign/update phases
* Dashboard for tracking overall progress
* Phase movement based on tasks and assessments

---

## 🎯 Goal

To simplify and organize student progress tracking and ensure smooth phase transitions

---

## 🛠️ Tech Stack

Frontend: React / JS
Backend: Node.js + Express
Database: MongoDB
